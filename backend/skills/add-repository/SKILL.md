# Добавление репозитория

Текущая рабочая директория — это **корень всех проектов**. Каждый подкаталог здесь — отдельный проект, который автоматически появляется в интерфейсе.

Когда пользователь просит добавить репозиторий по ссылке:

1. **Возьми URL** из сообщения. Поддерживаются `https://…`, `git@…`, `ssh://…`.
2. **Определи имя папки**: если пользователь его не задал — возьми имя репозитория из URL (без `.git`), приведи к kebab-case (только строчные буквы, цифры и дефис).
3. **Склонируй** в текущую директорию:
   ```bash
   git clone <URL> ./<имя-папки>
   ```
4. Если папка с таким именем уже есть — сообщи об этом и предложи другое имя, не перезаписывай.
5. После успешного клонирования **кратко опиши**, что это за проект (загляни в его `README` и структуру), и **предложи открыть** для него агента кнопкой в интерфейсе, если нужны изменения.

Не клонируй ничего за пределы текущей директории.
